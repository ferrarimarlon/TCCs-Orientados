## Objetivo do repositório

Este espaço reúne projetos acadêmicos orientados com foco em:

- Aplicação prática de conceitos de dados e inteligência artificial  
- Desenvolvimento de soluções reais para problemas de negócio  
- Formação de profissionais com mentalidade de engenharia  
- Documentação técnica e reprodutibilidade  

Os trabalhos priorizam **rigor técnico**, **clareza arquitetural** e **aplicabilidade prática**.

---

## Áreas de atuação

Os TCCs aqui presentes exploram temas como:

- Engenharia de Dados  
- Analytics Engineering  
- Machine Learning aplicado  
- Deep Learning  
- Processamento de Linguagem Natural (NLP)  
- Sistemas de recomendação  
- Business Analytics  
- Visualização de dados  
- Inteligência Artificial aplicada a negócios  

---

## Estrutura dos projetos

Cada TCC contém, quando aplicável:

- Descrição do problema
- Fundamentação teórica
- Metodologia
- Arquitetura da solução
- Código-fonte
- Dados (ou instruções de obtenção)
- Resultados e análises
- Documentação técnica
